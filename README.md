# 🎤 Serah Terima Jabatan
Naskah MC dan Doa untuk acara Serah Terima Jabatan

## 📝 Tentang Project
Project ini adalah website statis yang menampilkan naskah MC dan doa untuk acara serah terima jabatan. Website ini dibangun dengan HTML, CSS, dan JavaScript vanilla.

## 🎨 Fitur
- ✅ Interface yang responsif dan modern
- ✅ Navigasi smooth scroll
- ✅ Desain yang elegan dengan gradient
- ✅ Mobile-friendly
- ✅ Mudah di-customize

## 🚀 Deployment

### Option 1: GitHub Pages (Recommended - Free)
Website sudah siap di-deploy langsung ke GitHub Pages!

**Langkah-langkah:**

1. **Buka Repository Settings**
   - Go to: https://github.com/Wahyujs45/Serah-Terima-Jabatan/settings

2. **Aktifkan GitHub Pages**
   - Scroll ke bagian "Pages"
   - Pilih Branch: `main`
   - Pilih Folder: `/ (root)`
   - Klik Save

3. **Website Anda Live!**
   - Website akan tersedia di: https://wahyujs45.github.io/Serah-Terima-Jabatan/

### Option 2: Vercel (Lebih Cepat & Auto Deploy)

1. **Login ke Vercel**
   - Buka https://vercel.com

2. **Import GitHub Repository**
   - Klik "New Project"
   - Pilih GitHub Account
   - Cari "Serah-Terima-Jabatan"
   - Klik Import

3. **Deploy**
   - Vercel akan auto-deploy
   - Dapatkan URL custom

### Option 3: Netlify (Juga Gratis)

1. **Login ke Netlify**
   - Buka https://netlify.com

2. **Connect GitHub**
   - Drag & drop folder / Connect GitHub
   - Authorize Netlify

3. **Deploy**
   - Instantly live!

## 🎯 Cara Menggunakan

### Edit Naskah
Edit file `index.html` untuk mengubah konten naskah:

```html
<div class="scene">
    <h3>Nama Acara</h3>
    <p><strong>MC:</strong> Naskah Anda di sini</p>
</div>
```

### Ubah Warna
Edit file `styles.css` untuk mengubah warna:

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Tambah Halaman
Buat section baru di `index.html`:

```html
<section id="nama-section" class="section">
    <h2>Judul Section</h2>
    <div class="content">
        Konten di sini
    </div>
</section>
```
